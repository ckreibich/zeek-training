Solution
========
The participant is excepted to run the pcap through zeek , noting differing server banner strings and perhaps picking up on the old CTF favourite of morse code, being the reference to Samuel (Morse).

Note the server banners have implemented using the VersionAddendum directive in /etc/ssh/sshd_config. This allows for an additional text field to be tacked on the end of the banner in the form of "comment" as per https://datatracker.ietf.org/doc/html/rfc4253
The intent of this flag is to draw attention to the server and client banners, which are ostensible at best.
